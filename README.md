# CSS Tarea entregable 3 - Modern Art Gallery

Proyecto de dos vistas desarrollado con HTML5, CSS3 y JavaScript.

## Archivos

- `index.html`: página principal.
- `location.html`: página de ubicación.
- `css/styles.css`: estilos responsive.
- `js/script.js`: pequeña interacción visual con IntersectionObserver.
- `assets/`: imágenes extraídas del PDF de referencia.

## Cómo entregarlo

1. Sube todos los archivos a un repositorio de GitHub.
2. Activa GitHub Pages desde Settings > Pages.
3. Copia el enlace del proyecto publicado.
4. En la plataforma, pega el enlace del repositorio y el enlace de GitHub Pages.
